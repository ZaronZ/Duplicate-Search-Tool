# DST - Duplicate Search Tool


#### Usage
```
  -dir string
        base directory (default "./")
        
  -mask string
        file mask regexp (default ".*")
        
  -size-only
        size only comparison
```


#### Examples:   
```   
dst.exe -dir="D:\Files"
dst.exe -mask="(.*).txt"
dst.exe -size-only
```
